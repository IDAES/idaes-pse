^^^^^^^^^^^^^^^^^^^^^

**Install  Miniconda**

1. Download `Miniconda <https://repo.anaconda.com/miniconda/Miniconda3-latest-MacOSX-x86_64.sh>`_
2. Open a terminal window & run the downloaded script.
